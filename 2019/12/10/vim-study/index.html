学习了vim的一些操作

  • 移动 上 - k,下 - j, 左 - h, 右 - l
  • 进入插入模式 i（光标位置插入），I（行首插入），a（光标后插入），A（行末插入），o（向下另起一行），O（向上另起一行），s（删除当前光标所在位置插入），S（删除整行插入）
  • esc键退出插入模式，可视模式，可视块模式
  • 多行插入， ctrl + v 进入可视块模式（不是可视模式），然后大写i，进入插入，然后输入想要插入的内容，然后按两次esc即可
  • Vim-go 插件，在Vundle里面安装好之后还需要，还要执行 vim +GoInstallBinaries（下载一些二进制文件，这个过程基本上都是被qiang的，所以备好ti子），使用ctrl + ] 进行代码跳转，ctrl + t 返回
  • 使用vimtutor进行练习（很必要）
  • 使用ctrl + f（向下翻页），ctrl + b（向上翻页）
  • 使用ycm的时候，下载完成ycm之后需要进入源文件夹目录进行编译，然后再.vimrc中加入两行配置 这样就可以使用寻找函数使用的地方了
  • NERDTree 插件
    1. 快捷键ma，可以快速增加文件 m - 菜单， a - add a node
    2. 快捷键u,可以快速返回上级菜单，快捷键 C 可以快速进入目录
